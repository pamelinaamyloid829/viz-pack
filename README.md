## ✨ What viz-pack does

viz-pack helps you turn plain content into visuals you can share. It includes four Claude skills for making:

- diagrams
- infographics
- interactive explainers
- shareable dashboards

You type a simple command, and viz-pack guides the rest. It works well for notes, reports, plans, and other text you want to make easier to read.
